FX API Index

A prototype API to perform FX calculation with xrcodes e.g. (GBP, USD, EUR).

Fees are applied during conversion: amount < £1000 = -£25 | amount >= £1000 = -1%

GBP based rates used.

Version 1.0


Get rates: localhost:5000/api/v1/xrs

Get single rate: localhost:5000/api/v1/xrs/[xrcode]

Convert currency: localhost:5000/api/v1/xrs/[xrcode to exchange from]/[xrcode to exchange to]/[amount]

Convert currency with fee: localhost:5000/api/v1/xrs/[xrcode to exchange from]/[xrcode to exchange to]/[amount]?fee
